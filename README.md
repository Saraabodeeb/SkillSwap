## üìñ Project Overview
SkillSwap is a web-based platform designed to solve the problem of expensive education. It connects university students who want to exchange skills (e.g., "I teach you Python, you teach me Graphic Design") using a time-bartering system called "SkillCoins."

## ‚ú® Key Features
* **User Authentication:** Secure Login & Registration system using encryption (bcrypt).
* **Real-Time Global Chat:** A polling-based chat interface for community discussions (AJAX/Fetch API).
* **Dashboard:** Dynamic tracking of "SkillCoin" balances and learning history.
* **Database Integration:** Full MySQL persistence for Users, Transactions, and Messages.
